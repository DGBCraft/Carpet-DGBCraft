# Features


## permissionLevelSeed
Modify permission level of `/seed` command  
This rule does not affect the single player world  
* Type: `String`
* Default value: `2`
* Required options: `0`, `1`, `2`, `3`, `4`
* Categories: `COMMAND`

## sleepIgnoreFaker
Ignore fake players when skipping the night by sleeping  
* Type: `Boolean`
* Default value: `false`
* Required options: `true`, `false`
* Categories: `SURVIVAL`
